# CIFAR-10 Image Classification using CNN (PyTorch)

## 📌 Project Overview
This project implements a Convolutional Neural Network (CNN) using PyTorch to classify images from the CIFAR-10 dataset.

## 🚀 Features
- Data preprocessing using torchvision  
- CNN model with convolution + pooling layers  
- Training using Adam optimizer  
- Model evaluation on test dataset  

## 🛠️ Tech Stack
- Python  
- PyTorch  
- Torchvision  

## 📂 Dataset
CIFAR-10 dataset (automatically downloaded using torchvision)
